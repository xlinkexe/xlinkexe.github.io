Linux系统常用命令

· 2024-08-09 · # 服务器 # Linux

最近发现一些新买来的云服务器或多或少都有些使用不便，在此记录一些常用的Linux命令

free -h # 查看运行内存情况
df -h # 查看硬盘及其挂载情况
su # 切换到root用户

Debian

常用系统命令

sudo apt update # 软件包更新
sudo apt upgrade # 系统更新

换清华源(Debian11)

在命令行里输入

sudo vim /etc/apt/sources.list

按i，然后将文件中内容替换为如下代码

# 默认注释了源码镜像以提高 apt update 速度，如有需要可自行取消注释
deb https://mirrors.tuna.tsinghua.edu.cn/debian/ bullseye main contrib non-free
deb-src https://mirrors.tuna.tsinghua.edu.cn/debian/ bullseye main contrib non-free

deb https://mirrors.tuna.tsinghua.edu.cn/debian/ bullseye-updates main contrib non-free
deb-src https://mirrors.tuna.tsinghua.edu.cn/debian/ bullseye-updates main contrib non-free

deb https://mirrors.tuna.tsinghua.edu.cn/debian/ bullseye-backports main contrib non-free
deb-src https://mirrors.tuna.tsinghua.edu.cn/debian/ bullseye-backports main contrib non-free

# 以下安全更新软件源包含了官方源与镜像站配置，如有需要可自行修改注释切换
deb https://mirrors.tuna.tsinghua.edu.cn/debian-security bullseye-security main contrib non-free
deb-src https://mirrors.tuna.tsinghua.edu.cn/debian-security bullseye-security main contrib non-free

再按 ESC 由编辑模式退出到正常模式，接着输入 :wq 保存并退出
然后进行更新

sudo apt update
sudo apt upgrade

安装python

# 安装构建Python源代码所需的软件包
sudo apt install build-essential zlib1g-dev libncurses5-dev libgdbm-dev libnss3-dev libssl-dev libreadline-dev libffi-dev curl
# 使用wget或从Python下载页面下载最新版本的源代码curl
curl -O https://www.python.org/ftp/python/3.9.9/Python-3.9.9.tar.xz
# 下载完成后，解压缩tarball
tar -xf Python-3.9.9.tar.xz
# 导航到Python源目录并运行configure脚本
cd Python-3.9.9./configure --enable-optimizations
# 运行make以开始构建过程
make -j 4 # 修改 -j 使其与处理器中的内核数量相对应，可以通过键入找到号码nproc
# 构建完成后，以具有sudo访问权限的用户身份运行以下命令来安装Python二进制文件
sudo make altinstall
# 可以通过键入以下内容进行验证
python --version
